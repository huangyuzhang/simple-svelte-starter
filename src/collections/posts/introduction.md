This is a simple starter template for building a Svelte app with SvelteKit. It includes:

- Svelte + SvelteKit - PWA app
- TailwindCSS - styling
- Shadcn UI - UI components
- Typescript
- Paraglide.js - i18n
- Dark mode
- NProgress - progress bar during page redirects
- Responsive design
- Collections Management (markdown files)
  - blog posts
  - products
  - Tags: autogenerated from posts and products
  - Authors' profile pages
- Search Dialog
  - for posts & products
- Error pages
